# MagiWolf Quest — GDevelop → HTML5 (Chromebook-friendly)

Target: Chromebook. Play by opening `index.html` from the HTML5 export.

## Core
- Platform adventure (~30–60 minutes).
- Areas: Lodge Lobby → Water Park → Forest Trail → Magi Tower → Victory.
- Light “math gate” doors (kid arithmetic).
- Light “pet companion” perks (one active at a time).
- Player name: **Magi Alastair**.

## Controls
←/→ or A/D to move • Space/W/↑ to jump • E to interact • P to pause • 1–4 to answer quizzes.

## Build/Run
1) Use **GDevelop (web)** to create the project.  
2) Export to **HTML5** → place the export inside `export-html5/`.  
3) On Chromebook: open `export-html5/index.html` in Chrome.

## Branching
